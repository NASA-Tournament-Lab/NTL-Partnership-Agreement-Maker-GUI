Wireframe Notes

Rouing packages for My Review:

This sections lists all RPs that the user has to review. The reviews that are late are highlighted to bring them to the user's notice immediately.


Notifications:

This section lists all the notifications that the user maybe interested in and keeps them updated on what has happened since last they logged in.


My Packages:

This lists all the user's RPs, so that the user can keep track of them easily. By default the In Routing and Created RPs are displayed. The user can access their RP in a singe click from the homepage.


Browse RPs:

User can perform basic filtering of the RPs in a single click from here. For example, they can access all In Routing RPs for a specific group by clicking on the "In Routing(00)" link for the group . By clicking on the group name, they can access all RPs for that group in all states. The links also give the number of RPs in each category.
