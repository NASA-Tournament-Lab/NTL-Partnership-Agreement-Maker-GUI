Reviewers & Files Tab:

The reviewers list is similar to the current solution except for the below changes:


1. The parallel/serial (P/S) option is displayed together with the routing sequence. When changing the option between P and S, the routing seq # also gets updated.


2. Comments are a seperate column and they are easier to follow when there are to and fro comments between the admin and reviewers.


3. Email also has a seperate column with details on last sent mail and option to send email manually.


The user will be able to add reviewers, insert a routing template or save the current lost as a routing template etc.


 Files:

The usre can drag drop files in addition to browsing for them.


Distribution Options:

Allows user to add/edit distribution options


History:

Lists the history of the RP since it was created. It will allos user to see the timeline fo the package easily
